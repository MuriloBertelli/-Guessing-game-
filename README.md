# Jogo da Adivinhacao (C)

Este repositório contém dois programas em C que implementam um simples jogo de adivinhação:
- `FOR/advs.c` — versão simples (while), sem limite de tentativas.
- `while/while.C` — versão com níveis de dificuldade (for), com limite de tentativas.

## Como compilar

Use o `Makefile` fornecido:

```bash
make        # compila ambos os programas
make debug  # compila com simbolos de debug
make release# compila otimizado
